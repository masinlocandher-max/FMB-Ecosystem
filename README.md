# FMB Digital Ecosystem

This repository contains the independently branded websites and applications within the FMB ecosystem.

## Applications

- `apps/withlovefmb` - Francine Marie Bautista's personal website and official bulletin
- `apps/senz` - SENZ corporate website
- `apps/cognita` - Cognita Institute website and application

Each application remains isolated so it can retain its own domain, visual identity, deployment settings, and environment variables while being managed from one GitHub repository.
